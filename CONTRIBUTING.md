# Contributing

Thanks for helping keep this list awesome. A few guidelines so the bar stays high.

## What belongs here

- **Tools** that measurably help with GEO/AEO (auditing, structured data, measurement, crawl control).
- **Primary sources**: research papers, official vendor documentation, specifications.
- **Genuinely useful writing**: guides that teach a technique, not restate the obvious.

## What doesn't

- Link farms, thin listicles, or SEO spam.
- Paywalled content with no free value.
- Products with no GEO/AEO-specific angle (generic SEO suites, unless they ship a distinct AI-search feature).
- Self-promotion with no substance.

## How to add a link

1. Fork and create a branch.
2. Put the link in the most relevant section. Keep entries alphabetical where a section is already ordered that way.
3. Format: `` - [Name](https://url) — one line on *why it matters for GEO/AEO*. ``
   - Describe the **value**, not just the category. "Validates JSON-LD against Schema.org" beats "a schema tool".
4. One link per pull request unless they're closely related.
5. Run a quick check: no dead links, correct Markdown, no trailing whitespace.

## Style

- Sentence-case descriptions, ending with a period.
- Use `⭐` only for resources maintained by the list owner.
- Prefer canonical/official URLs over aggregators.

## Review

Maintainers may edit descriptions for consistency or ask for a stronger justification. A link that can't be described in terms of *why it helps someone get cited by AI search* probably doesn't belong yet.
